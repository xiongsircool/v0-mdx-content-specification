📚 学习资源 ​

为生物信息学学习者提供系统的学习资源，包括文章、视频、项目和数据集等。

🎯 推荐学习路径 ​

🌱 零基础入门（1-3个月） ​

  1. 🖥️ 基础技能

    • Linux 命令行基础
    • Python 编程入门
    • 生物学基础概念

  2. 📊 数据处理

    • 数据格式介绍（FASTA, FASTQ, BAM, VCF）
    • 基础统计分析
    • 数据可视化基础

  3. 🧬 初级分析

    • 序列比对基础
    • 基因注释简介
    • 简单的变异检测

🚀 进阶学习（3-6个月） ​

  1. 🔬 NGS数据分析

    • RNA-seq 完整流程
    • ChIP-seq 分析
    • ATAC-seq 数据处理

  2. 🧮 算法与工具

    • 常用算法原理
    • 高性能计算
    • 流程自动化

  3. 📈 高级统计

    • 多重检验校正
    • 机器学习应用
    • 网络分析基础

💻 实战项目（6个月+） ​

  1. 🎯 真实项目

    • 癌症基因组分析
    • 微生物组研究
    • 药物靶点预测

  2. 🔬 研究方向

    • 单细胞测序分析
    • 空间转录组学
    • 多组学整合分析

  3. 📝 论文复现

    • 经典论文重现
    • 新方法开发
    • 算法优化

📖 核心学习资源 ​

📚 必读书籍 ​

  • 📖 生物信息学导论作者：Mount 经典教材，适合零基础入门

  • 🧬 Genome-scale Algorithm Design作者：Compeau, Pevzner 算法导向，深入浅出

  • 🔬 Bioinformatics Data Skills作者：Vince Buffalo 实用技能为主，代码丰富

🎥 视频课程 ​

  • 🎓 Coursera: Bioinformatics SpecializationUC San Diego 系统性课程，涵盖基础算法

  • 🧬 edX: Genomic Data ScienceJohns Hopkins 专注于实际数据分析

  • 🔬 YouTube: Biostars免费教程 问题导向，实用性强

🛠️ 实用工具 ​

序列分析 ​

  • BLAST - 序列相似性搜索
  • Bowtie2 - 序列比对工具
  • BWA - 短序列比对
  • SAMtools - SAM/BAM文件处理

变异分析 ​

  • GATK - 变异检测工具包
  • FreeBayes - 贝叶斯变异检测
  • ANNOVAR - 变异注释

功能分析 ​

  • DAVID - 功能富集分析
  • Enrichr - 基因集分析
  • STRING - 蛋白质互作网络

🎯 实战项目 ​

🏃‍♂️ 入门项目 ​

  1. 新冠病毒基因组分析

    • 数据获取：NCBI SRA
    • 质量控制：FastQC
    • 序列比对：Minimap2
    • 系统发育：Nextstrain

  2. RNA-seq 差异表达分析

    • 数据预处理：Trimmomatic
    • 序列比对：HISAT2
    • 表达定量：featureCounts
    • 差异分析：DESeq2

🚀 进阶项目 ​

  1. 癌症体细胞突变分析

    • 变异检测：Mutect2
    • 注释分析：VEP
    • 通路分析：ReactomePA
    • 可视化：maftools

  2. 单细胞 RNA-seq 分析

    • 质控过滤：Seurat
    • 降维聚类：UMAP
    • 细胞类型注释：SingleR
    • 差异分析：MAST

📊 数据资源 ​

🧬 基因组数据库 ​

  • NCBI - 综合生物信息学数据库
  • Ensembl - 真核生物基因组注释
  • UCSC Genome Browser - 基因组可视化
  • GENCODE - 基因注释

📋 表达数据库 ​

  • GEO - 基因表达数据库
  • ArrayExpress - 功能基因组学数据
  • TCGA - 癌症基因组图谱
  • GTEx - 基因型-组织表达

🧪 其他资源 ​

  • KEGG - 通路数据库
  • Reactome - 通路分析
  • STRING - 蛋白质互作
  • DrugBank - 药物数据库
