Project Overview ​

This is a VitePress-based website for SBCBIOchina (上海生物信息学联盟). The site features:

  • Chinese language content focused on bioinformatics
  • Clean documentation-style layout optimized for GitHub Pages deployment
  • Markdown-based content management with VitePress
  • Custom theme with Chinese typography optimization
  • Automated deployment via GitHub Actions

Development Commands ​

bash
# Development server
npm run docs:dev

# Build for production
npm run docs:build

# Preview production build
npm run docs:preview

# Deploy to GitHub Pages
npm run deploy

Content Structure ​

Directory Organization ​

docs/
├── .vitepress/          # VitePress configuration
│   ├── config.ts       # Main configuration
│   └── theme/          # Custom theme
│       └── style.css   # Custom styles
├── announcements/      # Announcements section
├── posts/             # Blog posts
├── learn/             # Learning resources
├── about/             # About page
└── index.md           # Homepage

Content Types ​

  1. Announcements (docs/announcements/)

    • Community updates, events, recruitment
    • Tags: 活动, 招新, 比赛, 讲座, 通知
    • Simple markdown files with frontmatter

  2. Posts (docs/posts/)

    • Academic articles and technical sharing
    • Frontmatter: title, author, date, tags
    • Chinese academic content

  3. Learn Resources (docs/learn/)

    • Educational materials by difficulty level
    • Types: 文章, 视频, 项目, 数据集, 幻灯片
    • Levels: 入门, 进阶, 实战

Key Configuration ​

VitePress Configuration (docs/.vitepress/config.ts) ​

  • Chinese language support with zh-CN
  • GitHub Pages base path: /sbcbiochina/
  • Custom navigation and sidebar structure
  • SEO meta tags and social sharing
  • Local search functionality

Custom Theme ​

  • Brand colors: Blue theme (#3b82f6)
  • Chinese font optimization
  • Custom CSS for better typography
  • Gradient background on homepage

GitHub Actions Deployment ​

  • Automatic deployment on main branch push
  • Uses GitHub Pages for hosting
  • Build artifacts in .vitepress/dist/

Development Notes ​

Content Creation ​

  • Use standard Markdown with frontmatter
  • Chinese content should use Chinese punctuation
  • Images should be placed in public/ directory
  • Internal links use relative paths

Deployment ​

  • Project automatically deploys to GitHub Pages
  • URL will be: https://username.github.io/sbcbiochina/
  • Build process optimized for static hosting

Content Guidelines ​

  • Focus on bioinformatics and computational biology
  • Maintain academic tone and accuracy
  • Include proper citations and references
  • Use clear, accessible language for educational content
