上海生物信息学联盟 ​

上海生物信息学联盟官方网站，专注于生物信息学学习与交流的学术社区。

🚀 快速开始 ​

本地开发 ​

  1. 克隆仓库
bash
git clone https://github.com/yourusername/sbcbiochina.git
cd sbcbiochina
  1. 安装依赖
bash
npm install
  1. 启动开发服务器
bash
npm run docs:dev
  1. 打开浏览器访问
http://localhost:5173/

构建和部署 ​

bash
# 构建生产版本
npm run docs:build

# 预览构建结果
npm run docs:preview

# 部署到 GitHub Pages
npm run deploy

📝 内容管理 ​

添加新公告 ​

在 docs/announcements/ 目录下创建 Markdown 文件：

markdown
---
title: 公告标题
date: 2024-10-01
tags: [活动]
---

# 公告标题

公告内容...

添加新文章 ​

在 docs/posts/ 目录下创建 Markdown 文件：

markdown
---
title: 文章标题
author: 作者名
date: 2024-10-01
tags: [标签1, 标签2]
---

# 文章标题

文章内容...

添加学习资源 ​

在 docs/learn/ 目录下创建相应的 Markdown 文件。

🛠️ 技术栈 ​

  • 框架: Vue 3
  • 文档生成: VitePress
  • 部署: GitHub Pages
  • 样式: 自定义 CSS 主题
  • 构建: Vite

📁 项目结构 ​

├── docs/
│   ├── .vitepress/          # VitePress 配置
│   ├── announcements/      # 公告
│   ├── posts/             # 文章
│   ├── learn/             # 学习资源
│   ├── about/             # 关于
│   └── index.md           # 首页
├── public/
│   └── custom.css         # 自定义样式
├── .github/workflows/     # GitHub Actions
├── package.json
└── README.md
