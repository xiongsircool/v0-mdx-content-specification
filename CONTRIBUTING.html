贡献指南 ​

感谢你对上海生物信息学联盟网站的关注！以下是参与贡献的指南。

本地开发 ​

环境要求 ​

  • Node.js 18+
  • npm 或 yarn

安装和运行 ​

  1. 克隆仓库
bash
git clone https://github.com/yourusername/sbcbiochina.git
cd sbcbiochina
  1. 安装依赖
bash
npm install
  1. 启动开发服务器
bash
npm run docs:dev
  1. 构建生产版本
bash
npm run docs:build

提交规范 ​

分支命名 ​

  • 功能分支：feature/功能名称
  • 修复分支：fix/问题描述
  • 文档分支：docs/文档更新

提交信息格式 ​

类型(范围): 简短描述

详细描述（可选）

类型包括：

  • feat: 新功能
  • fix: 修复问题
  • docs: 文档更新
  • style: 代码格式化
  • refactor: 重构
  • test: 测试相关
  • chore: 构建或工具变动

代码风格 ​

  • 使用 2 个空格缩进
  • Markdown 文件使用中文标点
  • 保持文件命名的一致性
  • 图片资源放在 public/ 目录下

审核流程 ​

  1. Fork 项目
  2. 创建功能分支
  3. 提交更改
  4. 创建 Pull Request
  5. 等待审核和合并

问题反馈 ​

如果发现问题或有改进建议，请：

  1. 检查是否已有相关问题
  2. 创建新的 Issue，详细描述问题
  3. 提供复现步骤和环境信息
